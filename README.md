# OS Pusher Poll

Real time poll using Pusher, MongoDB, Node.js and Express.

## Setup
Edit the `scripts/set-vars.js` file and add your MongoDB URI and Pusher key info

## Install
`npm install`

## Run dev server localhost:8080
`npm run dev`

## Run start (production)
`npm start`
